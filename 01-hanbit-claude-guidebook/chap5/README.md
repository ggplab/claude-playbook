## 실습 데이터: seoul-apt-latest.csv

서울특별시 아파트 매매 실거래, **계약일 2025-07-01 ~ 2026-06-30 (최근 1년)**. 72,459건, 자치구 25개, UTF-8.

- 출처: 국토교통부 실거래가 공개시스템 조건별 자료제공 (https://rt.molit.go.kr) — 로그인 없이 내려받을 수 있는 공개 데이터입니다
- 정제: 해제(취소) 거래 제외, 핵심 값 결측 행 제거. 개인정보 없음
- 수집일 2026-07-14 기준이며 실거래 신고 기한(30일)으로 6월 하순 거래 일부는 이후 추가될 수 있어 출간 전 재수집 예정
- 최신 데이터가 필요하면 위 출처에서 직접 내려받아 같은 형식으로 바꿔 쓰면 됩니다

### 컬럼 사전

| 컬럼 | 설명 |
|------|------|
| `contract_ym` | 계약 연월 (YYYY-MM) |
| `contract_date` | 계약일 (YYYY-MM-DD) |
| `gu` | 자치구명 (예: 강남구) |
| `dong` | 법정동명 (예: 개포동) |
| `complex` | 아파트 단지명 |
| `area_m2` | 전용면적 (제곱미터, 소수 1자리) |
| `floor` | 층 |
| `price` | 거래금액 (만원, 정수) |
| `price_per_pyeong` | 평당가 (만원, 정수 = 거래금액 ÷ 전용 평수) |
