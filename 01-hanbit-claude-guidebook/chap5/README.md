## 실습 데이터: seoul-apt-latest.csv

서울특별시 아파트 **매매·전세·월세 통합 실거래**, **계약일 2025-07-01 ~ 2026-06-30 (동일 1년)**. 320,141건(매매 72,459 · 전세 129,649 · 월세 118,033), 자치구 25개, UTF-8.

- 출처: 국토교통부 실거래가 공개시스템 조건별 자료제공 (https://rt.molit.go.kr) — 로그인 없이 내려받을 수 있는 공개 데이터입니다 (매매·전월세 각각 별도 조건별 자료제공에서 같은 기간으로 수집 후 병합)
- 정제: 매매는 해제(취소) 거래 제외, 핵심 값 결측 행 제거. 전월세는 국토부 원본에 해제 필드가 없어 별도 제외 없음. 개인정보 없음
- 매매 수집일 2026-07-14, 전월세 수집일 2026-07-21 기준이며 실거래 신고 기한(30일)으로 6월 하순 거래 일부는 이후 추가될 수 있어 출간 전 재수집 예정
- 최신 데이터가 필요하면 위 출처에서 매매·전월세를 각각 내려받아 같은 형식으로 바꿔 쓰면 됩니다

### 컬럼 사전

| 컬럼 | 설명 |
|------|------|
| `contract_ym` | 계약 연월 (YYYY-MM) |
| `contract_date` | 계약일 (YYYY-MM-DD) |
| `gu` | 자치구명 (예: 강남구) |
| `dong` | 법정동명 (예: 개포동) |
| `complex` | 아파트 단지명 |
| `area_m2` | 전용면적 (제곱미터, 소수 1자리) |
| `floor` | 층 |
| `deal_type` | 거래유형 — `매매` / `전세` / `월세` |
| `price` | 매매 거래금액 (만원, 정수). **매매 행에서만 값 존재** |
| `deposit` | 보증금 (만원, 정수). **전세·월세 행에서만 값 존재** |
| `monthly_rent` | 월세금 (만원, 정수). **월세 행에서만 값 존재** (전세는 0이 아니라 빈 값) |
| `price_per_pyeong` | 평당가 (만원, 정수 = 거래금액 ÷ 전용 평수). **매매 행에서만 값 존재** |

거래유형별로 가격 구조가 달라(매매=거래금액, 전세=보증금, 월세=보증금+월세금) 해당 없는 컬럼은 빈 값으로 둔다.
