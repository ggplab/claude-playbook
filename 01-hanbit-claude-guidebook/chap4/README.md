# 4장. 컴퓨터를 꺼도 돌아가는 주가 수집기 만들기

클로드 데스크톱 앱의 **코워크**와 대화해 국내 주식 시세 수집기를 만들고, 완성된 시스템을 **구글 앱스 스크립트 + 구글 스프레드시트** 위에 올려 둡니다. 한 번 만들어 두면 클로드 없이, 내 컴퓨터가 꺼져 있어도 매일 자동으로 시세가 쌓입니다.

## 실습 자료

| 항목 | 내용 |
|------|------|
| 유즈케이스 | 구글 시트 + 앱스 스크립트로 삼성전자·SK하이닉스 시세를 평일 오후 4시 자동 수집 |
| 사용 기능 | 코워크로 PRD→코드 제작 · 커넥터로 시트 읽기 · 데스크톱 앱 스킬화 · Gmail 주간 리포트 |
| 산출물 | PRD 문서 → 붙여넣기용 앱스 스크립트 코드 → 설치·실행 가이드 |

## 파일

| 파일 | 설명 |
|------|------|
| [`chap4_PRD.md`](./chap4_PRD.md) | 무엇을 만들지 고정한 요구사항 정의서(PRD). 수집 규칙과 예외 처리 7개 규칙 포함 |
| [`code.gs`](./code.gs) | 구글 시트에 붙여넣는 앱스 스크립트 기준본. `[주식 수집기]` 메뉴·자동 실행·주간 리포트까지 포함 |
| [`gs_install.md`](./gs_install.md) | 코드 설치·실행 가이드(내가 직접 할 일만 정리) |
| [`apps-script-collector/`](./apps-script-collector) | 제작 과정을 굳힌 스킬. 주가 외 환율·날씨·경쟁사 가격에도 재사용 |

> AI는 같은 프롬프트에도 코드의 표현이 조금씩 달라질 수 있습니다. 기준은 코드의 생김새가 아니라 동작입니다. 본문을 따라 했는데 막히면, 이 폴더의 기준본 [`code.gs`](./code.gs)를 그대로 복사해 붙여넣어도 똑같이 동작합니다.

## 진행 흐름 (Plan · Build · Sustain)

| 단계 | 하는 일 |
|------|---------|
| **Plan** | 무엇을·어디서·어디에·언제 모을지 정하고, 수집 경로를 클로드에게 검토받기 |
| **Build** | PRD로 계획을 고정 → 앱스 스크립트 코드 생성 → 구글 시트에 설치 |
| **Sustain** | 커넥터로 클로드가 시트를 직접 읽게 연결 → 제작 과정을 스킬로 저장 → 스킬을 재사용해 Gmail 주간 리포트 추가 |

## apps-script-collector 스킬

코드를 모르는 사람도 "복사해서 붙여넣기만 하면 되는" 형태로, 구글 시트에 매일 정해진 시각에 데이터를 쌓는 자동 수집 시스템을 만들어 주는 스킬입니다. 클로드 데스크톱 앱에 등록해 사용합니다.

```
apps-script-collector/
├── SKILL.md                          ← 스킬 정의 (호출 조건·절차)
└── references/
    ├── prd_structure.md              ← PRD 6개 절 구성 규칙
    ├── prd_example.md                ← 삼성전자·SK하이닉스 기준 완성 PRD 예시
    ├── apps_script_template.gs       ← 붙여넣기용 앱스 스크립트 완성 템플릿
    └── install_guide_template.md     ← 설치·실행 가이드 템플릿
```

산출물 3종은 항상 **PRD → 앱스 스크립트 코드(.gs) → 설치 가이드** 순서로 생성되며, 세 문서가 서로 어긋나지 않도록 PRD의 예외 처리 7개 규칙이 코드에 100% 반영됩니다.

## 준비물

- 구글 계정 (앱스 스크립트·스프레드시트는 무료)
- 클로드 유료 플랜(Pro·Max·Team·Enterprise) — 제작 과정에서 코워크를 사용합니다. 완성된 수집기는 구글 계정만으로 돌아가며 클로드 구독과 무관하게 유지됩니다.
