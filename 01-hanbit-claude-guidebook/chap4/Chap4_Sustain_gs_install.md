# 설치·실행 가이드 (내가 할 일만)

코드는 준비돼 있습니다. 아래 6단계만 따라 하면 끝나고, 그 뒤로는 매일 오후 4시에 시세가 쌓이고 매주 월요일 오전 8시에 지난주 요약 메일이 옵니다.

1. **구글 시트 새로 만들기**
   구글 드라이브에서 빈 스프레드시트를 하나 만듭니다. 이름은 자유(예: "한국주식 시세수집"). 표는 미리 그릴 필요 없습니다 — 코드가 만들어 줍니다.

2. **표준 시간대 확인 (시각이 한국 시간이 되도록)**
   시트 메뉴 `파일 → 설정`에서 시간대를 **(GMT+09:00) 서울**로 맞추고 저장합니다. 그래야 "오후 4시"와 "월요일 오전 8시"가 한국 시간으로 동작합니다.

3. **코드 붙여넣기**
   시트에서 `확장 프로그램 → Apps Script`를 엽니다. 편집기에 원래 있던 내용을 지우고, 받은 `code.gs` 내용을 **전체 복사해서 붙여넣고 저장**(디스크 아이콘)합니다.

4. **시트로 돌아가 새로고침**
   브라우저에서 시트 탭을 새로고침하면, 상단 메뉴에 **`주식 수집기`**가 새로 생깁니다.

5. **① 최초 설정 한 번 누르기**
   `주식 수집기 → ① 최초 설정`을 누릅니다. 처음 한 번은 구글이 권한 승인을 요청하는데, 이번에는 **시트 접근**과 함께 **Gmail로 메일 보내기 권한**도 물어봅니다(주간 리포트 발송용). 본인 계정으로 **허용**해 주세요. 이 한 번으로 탭·헤더와 두 가지 자동 실행(매일 수집 + 매주 리포트)이 모두 만들어집니다.

6. **② 지금 수집 · ⑤ 리포트 지금 발송으로 확인**
   `② 지금 수집`을 눌러 종목 탭에 오늘 줄이 쌓이는지, `⑤ 리포트 지금 발송`을 눌러 내 Gmail로 요약 메일이 오는지 확인합니다. `로그` 탭에 "저장 완료", "발송 완료"가 남으면 정상입니다.

이후로는 손댈 필요 없이 매일 오후 4시에 수집하고, 매주 월요일 오전 8시에 지난주 요약 메일을 보냅니다.

---

## 메뉴 설명

- **① 최초 설정** — 탭·헤더를 만들고 자동 실행(매일 오후 4시 수집 + 매주 월요일 오전 8시 리포트)을 켭니다. 처음 한 번만 누르면 됩니다.
- **② 지금 수집** — 기다리지 않고 지금 즉시 오늘 시세를 한 줄 수집합니다.
- **③ 자동 실행 켜기 / 끄기** — 매일 수집과 매주 리포트를 함께 잠시 멈추거나 다시 켤 때.
- **④ 문제 진단** — 탭·자동 실행·리포트 주소·시세 통신이 정상인지 점검해 결과를 보여줍니다. 뭔가 이상하면 이걸 먼저 눌러 보세요.
- **⑤ 리포트 지금 발송** — 다음 월요일을 기다리지 않고, 지난주 요약 메일을 지금 바로 보냅니다.

## 알아두면 좋은 점

- **리포트 받는 주소**는 기본적으로 시트를 만든 본인 Gmail입니다. 다른 주소로 받으려면 코드 위쪽 `REPORT_EMAIL`에 주소를 넣으세요.
- 값이 이상하거나 비어 있을 때는 **`로그` 탭**을 보면 원인이 적혀 있습니다. 리포트 발송 성공/실패도 여기 남습니다.
- **주말·공휴일**과 **이미 저장된 날짜**는 자동으로 건너뜁니다. ① 최초 설정을 여러 번 눌러도 자동 실행이 중복으로 생기지 않습니다.
- **종목을 바꾸거나 늘리려면** 코드 위쪽 `SYMBOLS` 목록에 `{ name: '탭이름', code: 'KRX코드' }`를 추가하면 됩니다.
- **공휴일을 반영하려면** 코드 위쪽 `HOLIDAYS` 목록에 `'2026-01-01'` 형식으로 날짜를 넣으면 그날은 건너뜁니다.
