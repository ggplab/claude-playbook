# 국내 주식 시세 자동 수집기 — PRD

> **한 줄 요약:** 관심종목(예: 삼성전자, SK하이닉스)을 구글 시트에 등록해 두면, **장 마감 후 매일 자동으로** 그날의 시세(종가·시가·고가·저가·전일대비·등락률·거래량)와 날짜를 시트에 한 줄씩 쌓아 주는, **코드 없이 100% 무료**로 운영되는 데이터 수집기.

| 항목 | 내용 |
|---|---|
| 문서 버전 | v1.0 |
| 작성일 | 2026-07-09 |
| 대상 시장 | 국내 증시 (KOSPI / KOSDAQ) |
| 구현 방식 | Google Sheets + Google Apps Script (시간 기반 트리거) |
| 비용 | 무료 (Google 계정만 있으면 됨) |
| 코딩 필요 | 없음 — 제공된 스크립트를 복사·붙여넣기만 |

---

## 1. 배경 및 문제 정의

주식 투자자는 관심종목의 시세 변화를 꾸준히 기록해 두고 싶어 하지만, 다음과 같은 어려움이 있다.

- 증권사 앱은 **과거 시세를 내 방식대로 누적·가공**하기 어렵다.
- 매일 장 마감 후 직접 가격을 확인해 스프레드시트에 옮겨 적는 것은 **번거롭고 빠뜨리기 쉽다.**
- 유료 데이터 API나 서버를 쓰기에는 **비용과 기술 장벽**이 크다.

즉, **"관심종목만 등록해 두면, 사람이 손대지 않아도 매일 알아서 시세가 쌓이는"** 무료·무코딩 자동화가 필요하다.

---

## 2. 목표

### 2.1 제품 목표
- 사용자가 관심종목을 **한 번만 등록**하면, 이후에는 **아무 조작 없이** 매 거래일 시세가 자동으로 누적된다.
- 데이터는 **날짜 + 종목 + 시세 항목**의 시계열 형태로 시트에 쌓여, 언제든 조회·차트·분석이 가능하다.
- 전 과정이 **무료**이며 **서버·프로그램 설치가 필요 없다.**

### 2.2 성공 기준 (Acceptance Criteria)
1. 관심종목 시트에 종목코드를 추가하면 다음 거래일부터 자동으로 데이터가 쌓인다.
2. 매 거래일 장 마감 후 **1회**, 등록된 모든 종목의 시세가 데이터 시트에 각각 한 행씩 추가된다.
3. 주말·공휴일 등 **장이 열리지 않은 날에는 데이터가 중복/오기입되지 않는다.**
4. 기존에 쌓인 데이터는 **덮어써지지 않고 계속 아래로 누적**된다.
5. 사용자가 **코드를 이해하지 못해도** 문서의 안내만 따라 설치·운영할 수 있다.

### 2.3 비목표 (Out of Scope) — v1 제외
- 실시간(초 단위) 시세 스트리밍
- 매수/매도 자동 주문 등 트레이딩 기능
- 분·시간 단위 인트라데이 데이터 (v1은 **일 단위 1회** 종가 스냅샷)
- 해외 주식, 선물/옵션, 코인 (향후 확장 항목)

---

## 3. 사용자 & 사용 시나리오

**주 사용자 (페르소나):** 개인 투자자. 코드를 모르고, 구글 계정을 사용하며, 무료로 관심종목 시세 이력을 자동으로 모으고 싶어 한다.

**핵심 사용 시나리오:**
1. 사용자가 구글 시트의 `관심종목` 탭에 **삼성전자 / SK하이닉스**를 등록한다.
2. 이후 매 거래일 오후, 장이 끝나면 시스템이 스스로 동작한다.
3. 사용자는 저녁에 시트의 `데이터` 탭을 열어, 오늘 종가가 한 줄 추가된 것을 확인한다.
4. 며칠~몇 주가 지나면 종목별 시세 이력이 자동으로 쌓여 있고, 이를 그래프로 그리거나 다운로드해 분석한다.

## 4. 기능 요구사항

### 4.1 관심종목 관리
- 사용자는 `관심종목` 시트에 **종목명, 종목코드, 시장(KOSPI/KOSDAQ)** 을 입력해 종목을 추가·삭제할 수 있다.
- 종목 추가/삭제는 **행을 쓰거나 지우는 것만으로** 즉시 반영된다(별도 저장 버튼 불필요).
- 시스템은 관심종목 목록을 읽어, 등록된 모든 종목에 대해 수집을 수행한다.

### 4.2 자동 시세 수집 (핵심)
- 매 거래일 **장 마감 이후 지정된 시각(예: 16:00 KST)** 에 자동으로 1회 실행된다.
- 각 종목에 대해 아래 **수집 항목**을 가져와 `데이터` 시트에 **한 행씩 추가(append)** 한다.
- 실행 시각의 값은 정규장 종료(15:30) 이후이므로 **당일 종가 기준**이 된다.

### 4.3 수집 항목 (데이터 스키마)
`데이터` 시트에 아래 컬럼 순서로 누적한다.

| 컬럼 | 설명 | 데이터 출처(GOOGLEFINANCE 속성) |
|---|---|---|
| 날짜 | 수집 기준일 (YYYY-MM-DD) | 실행 시각 기준 자동 기록 |
| 종목명 | 예: 삼성전자 | 관심종목 시트 |
| 종목코드 | 예: 005930 | 관심종목 시트 |
| 종가(현재가) | 당일 종가 | `price` |
| 전일대비 | 전일 종가 대비 등락 금액 | `price` − `closeyesterday` |
| 등락률(%) | 전일 대비 등락률 | `changepct` |
| 시가 | 당일 시가 | `priceopen` |
| 고가 | 당일 고가 | `high` |
| 저가 | 당일 저가 | `low` |
| 거래량 | 당일 거래량 | `volume` |

> 참고: GOOGLEFINANCE는 KRX 종목을 `KRX:005930` 형태로 조회한다. 표시 가격은 약 **15~20분 지연 시세**일 수 있으나, 장 마감 후 수집하므로 **종가 확정값과 동일**하게 기록된다.

### 4.4 중복·휴장 처리
- **같은 날짜의 같은 종목 데이터가 두 번 쌓이지 않도록** 중복 방지 로직을 둔다(당일 이미 수집됨 → 건너뜀).
- **주말/공휴일 등 휴장일**에는 수집을 건너뛴다(가격이 갱신되지 않으므로 전일과 동일값 중복 방지).
- 수집 실패(값 없음/일시 오류) 시 해당 종목은 건너뛰고, 다음 종목 수집을 계속한다.

### 4.5 (선택) 알림 & 로그
- 수집 성공/실패 요약을 **본인 이메일로 발송**하거나 `로그` 시트에 기록하는 옵션(선택 기능).
- v1에서는 `로그` 시트 기록을 기본, 이메일 알림은 선택 사항으로 둔다.

## 5. 기술 아키텍처 (무료·무코딩)

```
┌────────────────────────────────────────────────────────────┐
│                      Google Sheets (문서 1개)                │
│                                                            │
│  [관심종목] 탭   →  종목명 / 종목코드 / 시장                    │
│  [데이터]   탭   →  날짜·종가·시가·고가·저가·등락률·거래량 누적    │
│  [로그]     탭   →  실행 이력(성공/실패)                        │
└───────────────▲──────────────────────┬─────────────────────┘
                │ 읽기                   │ 쓰기(append)
                │                       │
        ┌───────┴───────────────────────▼───────┐
        │        Google Apps Script (무료)        │
        │  - 관심종목 목록 읽기                     │
        │  - GOOGLEFINANCE로 시세 조회             │
        │  - 데이터 시트에 한 행 추가               │
        │  - 중복/휴장 체크                        │
        └───────────────▲───────────────────────┘
                        │ 매일 정해진 시각 자동 실행
                ┌───────┴────────┐
                │ 시간 기반 트리거  │  (Time-driven Trigger)
                │  매일 16:00 KST │
                └────────────────┘
```

### 5.1 구성 요소
| 요소 | 역할 | 비용 |
|---|---|---|
| **Google Sheets** | 데이터 저장소 + 사용자 화면 | 무료 |
| **Google Apps Script** | 수집 로직 실행 (시트 내장, 설치 불필요) | 무료 |
| **시간 기반 트리거(Trigger)** | 매일 정해진 시각에 스크립트 자동 실행 | 무료 |
| **GOOGLEFINANCE 함수** | 국내 종목 시세 조회 | 무료 |

### 5.2 데이터 소스 선택 근거
- **GOOGLEFINANCE**는 Google이 공식 제공하는 무료 함수로, `KRX:종목코드` 형식으로 국내 KOSPI/KOSDAQ 종목의 가격·시가·고가·저가·거래량·등락률을 조회할 수 있다.
- 별도 API 키·회원가입·유료 결제가 필요 없어 **"무료·무코딩"** 요구에 가장 부합한다.
- (대안: 네이버/증권사 크롤링 → 차단·구조 변경 위험이 커 v1에서는 제외.)

### 5.3 동작 흐름 (매 거래일)
1. **16:00 트리거 발동** → Apps Script 실행.
2. 오늘이 **주말/공휴일이면 종료**(로그만 기록).
3. `관심종목` 목록을 읽는다.
4. 종목별로 GOOGLEFINANCE 값을 조회한다(종가·시가·고가·저가·등락률·거래량).
5. `데이터` 시트에 **오늘 날짜 + 종목 + 시세**를 한 행 추가한다(이미 있으면 건너뜀).
6. 결과를 `로그` 시트에 기록(성공 n건 / 실패 n건).

## 7. 비기능 요구사항

- **무료**: 모든 구성 요소가 개인 Google 계정 한도 내에서 무료로 동작.
- **무인 운영**: 최초 설정 후 사용자의 개입 없이 매일 자동 실행.
- **안정성**: 특정 종목 조회 실패가 전체 수집을 중단시키지 않는다.
- **데이터 보존**: 기존 누적 데이터는 절대 삭제·덮어쓰기 하지 않는다.
- **가독성**: 코드 지식이 없는 사용자가 문서 안내만으로 설치·운영 가능.
- **확장성**: 관심종목을 늘려도 동일 로직으로 자동 처리(수십 종목 수준까지 무난).

## 8. 제약사항 및 리스크

| 구분 | 내용 | 대응 |
|---|---|---|
| 시세 지연 | GOOGLEFINANCE KRX 시세는 15~20분 지연될 수 있음 | 장 마감(15:30) 충분히 이후(예: 16:00)에 수집해 종가 확정값 확보 |
| 데이터 항목 한계 | 일부 속성(예: 특정 종목의 시가/거래량)이 값이 비어 올 수 있음 | 빈 값은 공란 처리, 로그에 기록 |
| 공휴일 캘린더 | 임시 공휴일 등 휴장일 자동 판별의 한계 | 전일과 종가가 동일하면 미갱신으로 간주해 스킵 또는 한국거래소 휴장일 목록 보정(향후) |
| 함수 정책 변경 | Google이 GOOGLEFINANCE 정책을 변경할 가능성 | 데이터 소스 교체 가능하도록 수집 로직 분리 |
| 실행 시각 정확도 | 트리거는 "정시 근처"에 실행(분 단위 오차 가능) | 시분 여유를 두어 마감 이후 시각으로 설정 |

## 9. 구현 단계 (마일스톤)

**1단계 — 시트 준비**
- 구글 시트 새 문서 생성 → `관심종목`, `데이터`, `로그` 3개 탭 구성.
- `관심종목`에 삼성전자(005930), SK하이닉스(000660) 입력.

**2단계 — 스크립트 설치**
- 시트 메뉴 `확장 프로그램 → Apps Script`에서 제공된 수집 스크립트를 붙여넣기.

**3단계 — 자동 실행 설정**
- Apps Script `트리거`에서 **매일 오후 4시(KST)** 시간 기반 트리거 등록.

**4단계 — 검증**
- 스크립트를 수동 1회 실행 → `데이터` 탭에 오늘 시세가 정상 추가되는지 확인.
- 다음 거래일 자동 실행 결과 확인.

**5단계 — 운영**
- 이후 관심종목만 관리하면 데이터가 자동 누적.

## 10. 향후 확장 (Backlog)

- 종가 급등락/목표가 도달 시 **이메일·카카오 알림**.
- 수집 데이터 기반 **자동 차트/대시보드** 시트.
- **이동평균·수익률** 등 파생 지표 자동 계산.
- 해외 주식·환율·지수 등 **수집 대상 확대**.
- 임시 공휴일까지 반영하는 **한국거래소 휴장일 연동**.

---

### 부록 A. 예시 GOOGLEFINANCE 사용
- 현재가: `=GOOGLEFINANCE("KRX:005930", "price")`
- 시가: `=GOOGLEFINANCE("KRX:005930", "priceopen")`
- 고가/저가: `=GOOGLEFINANCE("KRX:005930", "high")` / `"low"`
- 거래량: `=GOOGLEFINANCE("KRX:005930", "volume")`
- 등락률: `=GOOGLEFINANCE("KRX:005930", "changepct")`
- 전일 종가: `=GOOGLEFINANCE("KRX:005930", "closeyesterday")`
