# 설치·실행 가이드 (내가 할 일만)

코드는 준비돼 있습니다. 아래 6단계만 따라 하면 끝나고, 그 뒤로는 매일 오후 4시에 시트가 알아서 채워집니다.

1. **구글 시트 새로 만들기**
   구글 드라이브에서 빈 스프레드시트를 하나 만듭니다. 이름은 자유(예: "한국주식 시세수집"). 표는 미리 그릴 필요 없습니다 — 코드가 만들어 줍니다.

2. **표준 시간대 확인 (오후 4시가 한국 시간이 되도록)**
   시트 메뉴 `파일 → 설정`에서 시간대를 **(GMT+09:00) 서울**로 맞추고 저장합니다.

3. **코드 붙여넣기**
   시트에서 `확장 프로그램 → Apps Script`를 엽니다. 편집기에 원래 있던 내용을 지우고, 받은 `주식수집기.gs` 내용을 **전체 복사해서 붙여넣고 저장**(디스크 아이콘)합니다.

4. **시트로 돌아가 새로고침**
   브라우저에서 시트 탭을 새로고침하면, 상단 메뉴에 **`주식 수집기`**가 새로 생깁니다.

5. **① 최초 설정 한 번 누르기**
   `주식 수집기 → ① 최초 설정`을 누릅니다. 처음 한 번은 구글이 권한 승인을 요청합니다 — 본인 계정으로 **허용**해 주세요. 이 한 번으로 탭·헤더·자동 실행이 모두 만들어집니다.

6. **② 지금 수집으로 확인**
   `주식 수집기 → ② 지금 수집`을 눌러 바로 한 줄이 쌓이는지 봅니다. 삼성전자·SK하이닉스 탭에 오늘 줄이 생기고, `로그` 탭에 "저장 완료"가 남으면 정상입니다.

이후로는 손댈 필요 없이 매일 오후 4시에 자동으로 수집됩니다.

---

## 메뉴 설명

- **① 최초 설정** — 탭·헤더를 만들고 자동 실행(오후 4시)을 켭니다. 처음 한 번만 누르면 됩니다.
- **② 지금 수집** — 기다리지 않고 지금 즉시 오늘 시세를 한 줄 수집합니다.
- **③ 자동 실행 켜기 / 끄기** — 매일 자동 수집을 잠시 멈추거나 다시 켤 때.
- **④ 문제 진단** — 탭·자동 실행·시세 통신이 정상인지 점검해 결과를 보여줍니다. 뭔가 이상하면 이걸 먼저 눌러 보세요.

## 알아두면 좋은 점

- 값이 이상하거나 비어 있을 때는 **`로그` 탭**을 보면 원인이 적혀 있습니다.
- **주말·공휴일**과 **이미 저장된 날짜**는 자동으로 건너뜁니다. ① 최초 설정을 여러 번 눌러도 자동 실행이 중복으로 생기지 않습니다.
- **종목을 바꾸거나 늘리려면** 코드 위쪽 `SYMBOLS` 목록에 `{ name: '탭이름', code: 'KRX코드' }`를 추가하면 됩니다.
- **공휴일을 반영하려면** 코드 위쪽 `HOLIDAYS` 목록에 `'2026-01-01'` 형식으로 날짜를 넣으면 그날은 건너뜁니다.
