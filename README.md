# Claude Playbook

> 클로드, 어떻게 시작하고 어디까지 쓸 수 있을까. 시작부터 자동화까지 한곳에 정리합니다.

#Claude #ClaudeDesktop #ClaudeCode #Automation #NoCode #AI

## 이 페이지는

코딩을 모르는 직장인이 클로드(Claude)를 처음 켰을 때, "오, 이렇게 시작하면 되는구나"를 한 번에 훑을 수 있도록 정리한 입문 허브입니다. 가입·설치부터 핵심 기능, 유용한 스킬, CLI로 엮는 생산성 도구까지 한 흐름으로 안내합니다. 한빛미디어 〈클로드가 다 해줌〉(가제)의 실습 자료도 함께 제공합니다.

---

## 🚀 클로드 시작하기

코딩 없이, 데스크톱 앱을 켜고 말을 거는 것부터 시작합니다.

| 단계 | 할 일 | 비고 |
|------|------|------|
| 1. 계정 만들기 | [claude.ai](https://claude.ai) 접속 → 구글 또는 이메일로 가입 | 무료로 시작 가능 |
| 2. 데스크톱 앱 설치 | [claude.ai/download](https://claude.ai/download)에서 Mac·Windows 앱 다운로드 | 이 책 실습은 데스크톱 앱 기준 |
| 3. 로그인 | 설치한 앱에서 같은 계정으로 로그인 | — |
| 4. 첫 대화 | 입력창에 한국어로 질문을 적고 엔터 | 명령어 외울 필요 없음 |
| 5. (선택) 플랜 올리기 | 더 많이 쓰려면 Pro 구독 | 아래 [플랜 한눈에](#-플랜-한눈에) 참고 |

## 💳 플랜 한눈에

처음엔 무료로 충분합니다. 더 많이, 더 자주 쓰게 되면 Pro로 올리면 됩니다.

| 플랜 | 월 요금 | 한 줄 |
|------|--------|------|
| **Free** | $0 | 가볍게 체험해 보기 |
| **Pro** | $20 / 월 (연 $200) | 일상 업무용 — 대부분 여기서 충분 |
| **Max** | $100 (5×) ~ $200 (20×) | 하루 종일 헤비하게 쓰는 사람 |

> 요금은 2026년 5월 기준이며 변동될 수 있습니다. 최신 정보는 [claude.com/pricing](https://claude.com/pricing)에서 확인하세요. Max의 5×·20×는 Pro 대비 세션 사용량 배수입니다.

## 📑 기능 인덱스

클로드가 무엇까지 할 수 있는지 한눈에. 위에서 아래로 갈수록 더 자동화에 가깝습니다.

### 기본 작업 — 대화하며 바로 쓰는 것

| 기능 | 한 줄 |
|------|------|
| 프로젝트(Projects) | 관련 대화·파일·지침을 한 공간에 묶어 맥락을 유지 |
| 아티팩트(Artifacts) | 문서·코드·HTML을 대화 옆 캔버스로 만들고 바로 편집 |
| 메모리(Memory) | 이전 대화의 맥락을 기억해 이어서 활용 |
| 파일 첨부·입출력 | PDF·이미지·문서·스프레드시트를 올리고 결과물로 내려받기 |
| 음성 모드(Voice Mode) | 말로 입력하고 음성으로 대화 |
| 웹 검색(Web Search) | 최신 정보를 실시간으로 찾아 답변에 반영 |

### 연결·확장 — 외부 도구와 잇기

| 기능 | 한 줄 |
|------|------|
| MCP 커넥터(MCP Connectors) | Google Drive·Sheets·Notion 등 외부 서비스 연결 |
| 스킬(Skills) | 반복 작업 절차를 자산으로 만들어 자동 호출 |
| 코드 실행(Code Execution) | 파이썬·Node 샌드박스에서 코드를 직접 실행 |

### 자동화·자율 실행 — 더 나아가면

| 기능 | 한 줄 |
|------|------|
| Cowork | 데스크톱에서 멀티스텝 작업을 자율 실행 (권한 준 폴더 한정) |
| Routines | cron·API·GitHub 트리거로 클라우드에서 자율 실행 |
| Hooks | 라이프사이클 이벤트를 가로채 자동으로 명령 실행 |
| Computer Use | 마우스·키보드·화면을 직접 조작 (Pro·Max, 리서치 프리뷰) |
| Claude Code | 터미널·웹에서 코드베이스를 직접 다루는 CLI 에이전트 |

> 기능과 플랜별 제공 범위는 2026년 5~6월 기준이며 자주 바뀝니다. 최신 정보는 [공식 문서](https://support.claude.com)를 확인하세요.

## 🧩 유용한 스킬 모음집

**스킬(Skills)** 은 반복되는 작업 절차를 한 번 정의해 두면, 필요할 때 클로드가 알아서 꺼내 쓰는 기능입니다. "엑셀 정리해 줘", "SRT 예매해 줘" 같은 말 한마디에 맞는 스킬이 자동으로 불려 나옵니다.

### k-skill — 한국인을 위한 스킬 모음집

[NomaDamas/k-skill](https://github.com/NomaDamas/k-skill)은 한국 서비스에 맞춘 100개 이상의 AI 에이전트 스킬 모음집입니다. SRT·KTX 예매, 카카오톡, 주식·증권 정보, 부동산 검색, 쿠팡·다이소 쇼핑, 정부 민원처럼 손이 많이 가던 일을 클로드(및 Claude Code·Codex·OpenCode)가 대신 처리하게 해 줍니다.

> "한국인인가요? 이 스킬 모음집을 다운로드 받아 두세요. 언젠가 무조건 쓸 때가 옵니다!"

설치 후 `k-skill-setup`으로 자격 증명을 설정하면 바로 쓸 수 있습니다. 자세한 사용법은 [저장소](https://github.com/NomaDamas/k-skill)를 참고하세요.

### 클로드 기본 탑재 스킬

설치 없이 바로 쓰는 공식 스킬도 있습니다. Word(docx)·PDF·PowerPoint(pptx)·Excel(xlsx) 같은 문서 작업을 말로 시키면 알아서 처리합니다.

### 나만의 스킬 만들기

내가 매번 반복하는 셋업·정리 절차를 슬래시 명령 하나로 묶을 수 있습니다. 복사해서 바로 따라 만드는 예제를 모았습니다.

| 스킬 | 한 줄 | 가이드 |
|------|------|--------|
| `/new-project` | 새 프로젝트를 일관된 네이밍·git 셋업으로 시작 | [바로가기](./02-skills/new-project/) |

> 전체 목록과 스킬 구조 설명은 [스킬 모음](./02-skills/)에서 확인하세요.

## 🛠️ CLI 연동 생산성 도구

여기서부터는 살짝 심화입니다. **CLI**(명령줄)는 마우스 클릭 대신 터미널에 명령어를 입력해 쓰는 방식으로, 클로드를 더 깊이 부려 자동화를 본격적으로 하고 싶을 때 들어섭니다.

### Claude Code — 터미널 속 클로드

[Claude Code](https://claude.com/claude-code)는 Anthropic 공식 CLI 에이전트입니다. 터미널에서 코드베이스를 직접 읽고 고치고, 파일을 만들고, 명령을 실행합니다. Pro·Max 구독으로도 사용할 수 있고, 데스크톱 앱에서 한 단계 더 나아가고 싶을 때 자연스럽게 이어집니다.

### 그 밖의 코딩 에이전트

Codex, OpenCode 같은 다른 CLI 에이전트도 있습니다. 앞서 소개한 [k-skill](#k-skill--한국인을-위한-스킬-모음집)은 이들에서도 함께 동작합니다.

### MCP로 잇는 외부 도구

Notion·Supabase·Playwright 등을 MCP 커넥터로 연결하면, 클로드와 외부 서비스를 한 흐름으로 다룰 수 있습니다.

_각 도구의 설치·연동법은 추후 추가됩니다._
